# World Pulse Glossary

## Acronyms & Shorthand
- **WP**: World Pulse (this project)
- **CLTR**: Cut the Rope - Remastered (Kai's game project)
- **Sole AT**: Another pinned project of Kai's
- **TVL**: Total Value Locked (DeFi metric)
- **F&G**: Fear & Greed Index (crypto sentiment)
- **BTC Dom**: Bitcoin Dominance percentage

## Signal Strengths
- **Critical** (score >= 100): Immediate attention needed, multi-source convergence
- **Strong** (score >= 70): Notable signal with solid source diversity
- **Moderate** (score >= 40): Worth monitoring
- **Weak** (score < 40): Background noise

## Verticals
- **AI**: Artificial intelligence, ML, LLMs, compute
- **DeFi**: Decentralized finance, yields, protocols, TVL
- **Payments**: Payment rails, stablecoins, merchant solutions
- **Merchant**: Commerce, retail, distribution
- **Institutional**: Enterprise, regulatory, macro

## Data Sources
- **paperswithcode**: Academic ML/AI papers with code
- **hackernews**: Tech community discussion (Y Combinator)
- **github**: Trending repos and developer activity
- **reddit**: Subreddit signals (r/cryptocurrency, r/technology, etc.)
- **coingecko**: Crypto market data and prices
- **defillama**: DeFi protocol TVL and yields
- **polymarket**: Prediction market probabilities
- **google-trends**: Search trend signals

## Architecture Terms
- **PulseSnapshot**: The master data object from a single refresh
- **Cluster**: A group of related signals from multiple sources
- **Flow**: Capital or macro data movement (TVL, FX, commodities)
- **Entity**: Extracted named entity (company, token, tech, person)
- **WorldMap**: Interactive SVG visualization of source network
